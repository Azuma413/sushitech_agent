# プロダクト文脈（productContext.md）

## なぜこのプロジェクトが必要か
Unityアプリケーションと連携し、ユーザーの音声・テキスト入力をリアルタイムで処理できる柔軟な対話エージェントが求められている。既存のフレームワークは複雑で拡張性に乏しいため、シンプルかつ拡張しやすいバックエンドが必要。

## 解決する課題
- 音声・テキスト入力のリアルタイム処理
- 感情やツール制御を含む構造化応答の生成
- Unityとのスムーズな連携
- 旧実装の複雑さ・保守性の課題

## 期待されるユーザー体験
- 入力に即応する自然な対話
- エージェントの感情やツール利用が明確に伝わる
- Unityアプリへの容易な組み込みと拡張
